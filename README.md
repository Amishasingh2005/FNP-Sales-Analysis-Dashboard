# 📊 FNP Sales Analysis Dashboard (Microsoft Excel)

## 📌 Project Overview

The **FNP Sales Analysis Dashboard** is an interactive dashboard built using **Microsoft Excel** to analyze sales performance and generate meaningful business insights. The dashboard provides a comprehensive view of revenue, orders, customer spending, product performance, city-wise sales, and delivery metrics through dynamic visualizations and interactive filters.

---

## 🎯 Project Objective

The objective of this project is to:

- Analyze overall sales performance.
- Monitor key business KPIs.
- Identify top-performing occasions, categories, and products.
- Analyze customer purchasing behavior.
- Track monthly sales trends.
- Support data-driven business decisions.

---

## 📈 Dashboard KPIs

- 💰 **Total Revenue:** ₹3,975,424
- 📦 **Total Orders:** 1,000
- 👤 **Average Customer Spend:** ₹3,975.42
- 🚚 **Average Delivery Time:** 5.53 Days

## 📊 Dashboard Features

- Revenue by Occasion
- Revenue by Category
- Top 10 Cities by Orders
- Revenue by Months

- Revenue by Order Time

- Timeline Filters

## 💡 Key Business Insights

- Anniversary generated the highest revenue.
- Colors was the highest-performing product category.
- Sales showed seasonal peaks across different months.
- A few products contributed significantly to total revenue.
- Certain cities generated higher order volumes.
- Average order delivery time was approximately **5.5 days**.
